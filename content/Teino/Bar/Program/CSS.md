## display
要素の並べ方を指定するプロパティ。
通常要素は縦に積まれていく。

- block
    - 横いっぱいに広げ、縦に積む
- inline
    - 横に並べる
    - 縦横は内容依存
- inline-block
    - 横に並べる
    - 縦横を自分で決められる
- none
    - 非表示
- flex
    - 縦を揃え、横をページに合わせて配置

その他grid,ruby（読み仮名）,tableなどがある。

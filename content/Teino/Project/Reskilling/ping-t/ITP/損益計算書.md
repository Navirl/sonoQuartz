表はA社の期末の損益計算書から抜粋した資料である。当期純利益が800百万円であるとき，販売費及び一般管理費は何百万円か。
![65607.jpg](https://ping-t-resouces.com/uploads/question_image/file/26486/65607.jpg?t=1716873028)
?
A. 1000

損益計算書とは、一定期間の企業の経営成績を表す財務諸表の一つです。企業の収益と費用を記載し、最終的な利益を算出します。

当期純利益は、収益から全ての費用を差し引いた後の利益のことで、企業の最終的な儲けを表します。

販売費及び一般管理費は、商品の販売活動や企業の管理業務に関連する費用のことです。例えば、広告宣伝費、従業員の給料、事務所の家賃などが含まれます。

以下の表を使って、設問の解き方を説明します。

|項目|金額（百万円）|
|---|---|
|売上高|8,000|
|売上原価|6,000|
|販売費及び一般管理費|?|
|営業外収益|150|
|営業外費用|50|
|特別利益|60|
|特別損失|10|
|法人税等|350|
|当期純利益|800|

当期純利益は、以下の式で計算されます。

```
当期純利益 = 売上高 - 売上原価 - 販売費及び一般管理費 + 営業外収益 - 営業外費用 + 特別利益 - 特別損失 - 法人税等
```

この式に表の数値を当てはめると、

```
800 = 8,000 - 6,000 - 販売費及び一般管理費 + 150 - 50 + 60 - 10 - 350
```

となります。この式を販売費及び一般管理費について解くと、

```
販売費及び一般管理費 = 8,000 - 6,000 - 800 + 150 - 50 + 60 - 10 - 350 = 1,000
```

となります。

以上より、正解は「1,000」百万円となります。

別解：
以下のように、**売上総利益**、**営業利益**、**経常利益**、**税引前当期純利益**、**当期純利益**を追加した表を使って解く方法もあります。

|項目|金額（百万円）|
|---|---|
|売上高|8,000|
|売上原価|6,000|
|**売上総利益**|2,000|
|販売費及び一般管理費|?|
|**営業利益**|1,000|
|営業外収益|150|
|営業外費用|50|
|**経常利益**|1,100|
|特別利益|60|
|特別損失|10|
|**税引前当期純利益**|1,150|
|法人税等|350|
|**当期純利益**|800|

**売上総利益**は売上高から売上原価を引いたもの、**営業利益**は売上総利益から販売費及び一般管理費を引いたもの、**経常利益**は営業利益に営業外収益を加え営業外費用を引いたもの、**税引前当期純利益**は経常利益に特別利益を加え特別損失を引いたもの、**当期純利益**は税引前当期純利益から法人税等を引いたものです。

当期純利益が800百万円であることから、
税引前当期純利益 = 当期純利益 + 法人税等 = 800 + 350 = 1,150百万円
経常利益 = 税引前当期純利益 - 特別利益 + 特別損失 = 1,150 - 60 + 10 = 1,100百万円
営業利益 = 経常利益 - 営業外収益 + 営業外費用 = 1,100 - 150 + 50 = 1,000百万円

営業利益 = 売上総利益 - 販売費及び一般管理費より、
1,000 = 2,000 - 販売費及び一般管理費
したがって、販売費及び一般管理費は1,000百万円となります。

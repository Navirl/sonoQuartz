CSharpの[csharp ジェネリックメソッド](<./csharp ジェネリックメソッド.md>)と似た機能。
リアルタイムでソースを変更する機能っぽい？

```cpp
template <typename T>
T Sum(T a, T b) {
    return a + b;
}
```

あちらと違い、引数型の指定部分は先に出てくる。
typenameでもclassでも使える。
参照は参照型だからこのまま渡される、&は必要無い？

また`decltype`を使用することで、引数型から戻り値型を推論できる。

```cpp
template <typename T, typename U>
auto Sum(T a, U b) -> decltype(a + b) {
    return a + b;
}
```

応用で[可変長引数テンプレート](<./可変長引数テンプレート.md>)がある。
